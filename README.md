# CAS Operations

Scripts to run CAS analytics tasks

See the 'help' text in the Makefile for the most up to date instructions;

```
make help
```

## Usage

Clone this repo:

```
git clone --recurse git@github.com:mskcc/cas-ops.git
cd cas-ops
```

Run pipeline commands in dir:

```
cd tempo
make <some_command>
```
